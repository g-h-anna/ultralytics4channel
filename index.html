ultralytics4channel

Ultralytics 8.2.5 modified to train and predict for 4-channel (RGBD) images.
https://github.com/g-h-anna/ultralytics4channel

In this repository I modified the source code of Ultralytics to allow training and prediction for 4-channel images. Main modifications are:

  • Changing cv2.imread() functions to include cv2.IMREAD_UNCHANGED
  • Changing how the mosaic is generated for logging (plotting)
  • Changing every variable for channel number from 3 to 4

You find the steps in the pdf file included in the repository.
